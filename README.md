# Neverwinter Vault Issues

This repository is for reporting issues with [neverwintervault.org](https://neverwintervault.org).

## How to Report an Issue

When reporting an issue, please provide the following information to help us resolve it quickly:

### Required Information

1. **Clear Title**: Use a descriptive title that summarizes the issue
2. **Description**: Provide a detailed description of the problem
3. **Steps to Reproduce**: List the exact steps needed to reproduce the issue
4. **Expected Behavior**: Describe what you expected to happen
5. **Actual Behavior**: Describe what actually happened
6. **Browser and OS**: Specify your browser (name and version) and operating system

### Optional but Helpful Information

- **Screenshots**: If applicable, add screenshots to help explain the problem
- **Console Errors**: Include any browser console errors (F12 â†’ Console tab)
- **Network Issues**: Mention if you're experiencing connectivity problems
- **Account Type**: Specify if the issue occurs with a logged-in or guest account

## Issue Types

### Bug Reports
Use this for reporting problems with the website functionality, broken links, or unexpected behavior.

### Feature Requests
Use this to suggest new features or improvements to the Neverwinter Vault.

### Content Issues
Report problems with specific vault content, such as:
- Broken download links
- Missing or incorrect metadata
- Inappropriate content

### Account Issues
For issues related to user accounts, login, or permissions.

## Before Submitting

Please check if your issue has already been reported:
1. Search existing issues to avoid duplicates
2. Check if the issue persists in a different browser
3. Clear your browser cache and cookies to rule out caching issues

## Response Time

We'll do our best to respond to your issue as quickly as possible. Please be patient as we work through reported issues.

## Contributing

If you're interested in contributing to the Neverwinter Vault, please reach out through the issues section or contact the maintainers directly.
